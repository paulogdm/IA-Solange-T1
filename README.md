##Como adicionar mapas
O mapa deve ter pelo menos 2 pixels RGB:
Um ponto de start com a cor (0,255,0) FULLGREEN
Um ponto de fim com a cor (255,0,0) FULLRED
Paredes podem ser adicionadas como pixeis pretos
